## Industry Terminology Glossary

Include these terms naturally in FAQ answers:

| Term | Definition |
|------|------------|
| GGR | Gross Gaming Revenue - total bets minus winnings paid |
| NGR | Net Gaming Revenue - GGR minus bonuses and taxes |
| PAM | Player Account Management system |
| PSP | Payment Service Provider |
| RNG | Random Number Generator |
| RTP | Return to Player percentage |
| LTV | Lifetime Value of a player |
| CAC | Customer Acquisition Cost |
| CPA | Cost Per Acquisition (affiliate) |
| KYC | Know Your Customer verification |
| AML | Anti-Money Laundering compliance |
| MGA | Malta Gaming Authority |
| UKGC | UK Gambling Commission |
| B2B | Business to Business |
| B2C | Business to Consumer |
| API | Application Programming Interface |
| SLA | Service Level Agreement |
